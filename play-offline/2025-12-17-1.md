# Offline Play
Offline play is not in any way endorsed by mojang. That being said it falls under a gray area. However, offline play is permitted by the azox network.

## How to play offline

### Prism Launcher
Prism launcher is personally my favorite launcher
This is how to bypass authentication and use it while not owning an online account.
1. Download Prism from [prismlauncher.org]
2. Open Prism and go through quick setup
3. Close/Quit Prism
4. Go to [https://github.com/antunnitraj/Prism-Launcher-PolyMC-Offline-Bypass]
5. Run command for your OS
6. Open Prism again
7. DO NOT DELTE "No Profile" at any point
8. Go under accounts > manage accounts ...
9. Click on "add offline"
10. Input desired username
11. Create an installation from the main window
12. Install mods if Desired
13. Launch installation and play!

### Playing the server
1. Launch Installation
2. Select Multiplayer
3. Add server
4. Server name: azox (or any other)
5. Server IP: azox.net
6. Add server
7. Click and click join server
8. run /register YourPassword YourPassword (but replace with actual password)
9. Play!
